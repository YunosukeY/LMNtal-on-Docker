# LMNtal-on-Docker
## 使い方(Ubuntu)
1. ウィンドウの転送を許可
```Shell
$ xhost +local:lmntal-host
```

1. コンテナをビルド・起動
```
$ docker-compose up
```

1. 言語を選択  
日本語は文字化けします
